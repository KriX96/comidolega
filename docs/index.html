CoMiDolega

Opis projektu

Projekt CoMiDolega z założenia miał pomagać użytkownikowi w weryfikacji jego stanu zdrowia. Na podstawie symptomów określamy rodzaj choroby oraz dajemy wskazówki dotyczące leczenia. Aplikacja w obecnej fazie ma podstawowe funkcjonalności takie jak: Rejestracja, Logowanie, Sprawdzenie stanu zdrowia - zaznaczamy symptomy choroby które za pomocą API wysyłamy do naszego serwera. W frameworku Symfony 3 stworzyliśmy system (API) który analizuje przesłane do niego informacje i na podstawie symptomów dopasowuje najbardziej zbliżoną chorobę (lub jej brak jeżeli nie uda mu się znaleźć konkretnych połączeń między symptomami)
